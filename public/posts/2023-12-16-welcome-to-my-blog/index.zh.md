# 欢迎来到我的技术博客

大家好！这是我的第一篇博客文章。

## 关于这个博客

这个博客使用以下技术栈构建：

- **React 18** - 现代化的前端框架
- **TypeScript** - 类型安全的 JavaScript
- **Material-UI (MUI)** - 优雅的 UI 组件库
- **Vite** - 快速的构建工具
- **React Router** - 客户端路由
- **i18next** - 国际化支持

## 主要功能

### 1. 深色/浅色主题切换

你可以点击右上角的主题图标来切换深色和浅色主题。

### 2. 中英文切换

博客支持中英文双语，可以通过语言切换按钮进行切换。

### 3. Markdown 支持

所有文章都使用 Markdown 格式编写，支持：

- **粗体** 和 *斜体*
- 代码高亮
- 列表
- 引用
- 表格

### 4. 代码高亮

支持多种编程语言的语法高亮：

\`\`\`javascript
function greet(name) {
  console.log(\`Hello, \${name}!\`);
}

greet('World');
\`\`\`

\`\`\`python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(10))
\`\`\`

### 5. 文章分类和标签

文章可以按分类和标签进行筛选，方便查找相关内容。
